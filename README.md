# ChackCheck
책 관리 홈페이지입니다. 

읽은 책에 대한 서평을 정리하고 관심 책을 담을 수 있습니다. 

사용자들이 많이 읽은 트렌드한 책 정보와 활동로그를 확인할 수 있습니다.
